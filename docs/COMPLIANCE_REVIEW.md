# 合规文案评审

UseChat 是本地优先的消息软件 Computer Use 项目，面向个人助理、单账号、可见桌面、用户授权和可审计自动化场景。

## 允许表达

- 读取指定对话当前可见消息。
- 总结消息内容。
- 草拟回复。
- 在用户确认或明确授权后发送消息。
- 作为本地 Agent 工具连接用户自己的桌面消息软件。

## 禁止表达

- 防封、规避风控、绕过平台规则。
- 群发营销、多账号群控、批量私信。
- 破解协议、注入客户端、读取本地数据库。
- 隐藏发送行为或绕过用户授权。

## 文档扫描

当前文档允许出现“群发 / 营销”等词作为非目标或禁止用途说明，不作为卖点或操作指南。公开文档不提供防封、营销自动化或规避规则教程。

## 当前结论

当前公开文档口径应强调本地授权、可见桌面、人类控制、fail closed 和隐私最小化。正式公开前仍需产品 / 法务 / 安全最终复核。
